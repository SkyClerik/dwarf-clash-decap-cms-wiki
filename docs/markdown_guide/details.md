# Сворачивающиеся блоки (Details)

**Details** — это расширение, которое позволяет скрывать объемную или менее важную информацию, которую читатель может развернуть по желанию. Это очень удобно для секций "Часто задаваемые вопросы", "Подробности реализации" или "Примеры кода".

---

### Как использовать

Вы начинаете блок с `???` (три вопросительных знака), затем пишете заголовок (который будет виден в свернутом состоянии). Весь скрытый текст должен иметь отступ в четыре пробела.

Если вы хотите, чтобы блок был развернут по умолчанию, используйте `???+`.

### Примеры

#### Свернутый блок

**Шаблон:**
```markdown
??? "Нажмите, чтобы увидеть подробности"
    Это скрытый текст. Он будет виден только после того, как вы нажмете на заголовок "Нажмите, чтобы увидеть подробности".
    Здесь может быть длинное объяснение или пример кода.
```

**Результат:**
??? "Нажмите, чтобы увидеть подробности"
    Это скрытый текст. Он будет виден только после того, как вы нажмете на заголовок "Нажмите, чтобы увидеть подробности".
    Здесь может быть длинное объяснение или пример кода.

---

#### Развернутый по умолчанию блок

**Шаблон:**
```markdown
???+ "Этот блок изначально развернут"
    Этот текст виден сразу, но его можно свернуть, нажав на заголовок.
```

**Результат:**
???+ "Этот блок изначально развернут"
    Этот текст виден сразу, но его можно свернуть, нажав на заголовок.

---

#### Вложенные блоки

Вы можете вкладывать блоки друг в друга, а также комбинировать их с другими расширениями, например, с `Admonitions`.

**Шаблон:**
```markdown
??? question "Можно ли вкладывать блоки?"
    Да, конечно!

    ??? note "Пример вложенности"
        Здесь находится вложенный блок, который тоже можно сворачивать.
```

**Результат:**
??? question "Можно ли вкладывать блоки?"
    Да, конечно!

    ??? note "Пример вложенности"
        Здесь находится вложенный блок, который тоже можно сворачивать.
