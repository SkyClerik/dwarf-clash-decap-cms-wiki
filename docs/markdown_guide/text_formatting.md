# Форматирование текста и списки задач

Здесь собраны различные расширения для улучшения внешнего вида текста: списки задач, иконки, эмодзи и выделение.

---

### Списки задач (Tasklists)

Это расширение позволяет создавать списки с кликабельными чекбоксами.

**Шаблон:**
```markdown
- [x] Завершить настройку MkDocs
- [ ] Написать документацию
- [ ] Выпить чаю
```

**Результат:**
- [x] Завершить настройку MkDocs
- [ ] Написать документацию
- [ ] Выпить чаю

---

### Иконки и Эмодзи (Icons, Emojis)

Вы можете легко вставлять тысячи иконок и эмодзи прямо в текст.

**Как использовать:**
Просто напишите короткий код эмодзи, окруженный двоеточиями. Например, `:smile:`. Вы также можете использовать иконки из набора Material Design, например, `:material-home:`.

**Шаблон:**
```markdown
Я люблю Markdown! :partying_face:
Перейти на главную :material-home:
Это просто :fire:
```

**Результат:**
Я люблю Markdown! :partying_face:
Перейти на главную :material-home:
Это просто :fire:

---

### Выделение и форматирование текста

#### Выделение маркером (Mark)

**Шаблон:**
```markdown
Этот текст будет ==выделен маркером==.
```

**Результат:**
Этот текст будет ==выделен маркером==.

#### Зачеркивание (Tilde)

**Шаблон:**
```markdown
Этот текст будет ~~зачеркнут~~.
```

**Результат:**
Этот текст будет ~~зачеркнут~~.

#### Вставка (Caret)

**Шаблон:**
```markdown
Этот текст будет ^^вставлен^^.
```

**Результат:**
Этот текст будет ^^вставлен^^.

#### Отображение клавиш (Keys)

**Шаблон:**
```markdown
Нажмите ++ctrl+alt+del++ для перезагрузки.
```

**Результат:**
Нажмите ++ctrl+alt+del++ для перезагрузки.
