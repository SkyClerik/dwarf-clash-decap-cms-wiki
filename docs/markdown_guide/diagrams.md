# Диаграммы и графики (Mermaid)

**Mermaid** — это расширение, которое позволяет создавать сложные диаграммы и графики, используя текстовое описание. Это невероятно мощный инструмент для визуализации процессов, структур и данных.

---

### Как использовать

Вы создаете блок кода и указываете язык `mermaid`. Внутри этого блока вы пишете код, описывающий вашу диаграмму.

### Примеры

#### Блок-схема (Flowchart)

**Шаблон:**
```mermaid
graph TD
    A[Начало] --> B(Процесс);
    B --> C{Решение};
    C -->|Да| D[Конец];
    C -->|Нет| E[Другой процесс];
    E --> B;
```

**Результат:**
```mermaid
graph TD
    A[Начало] --> B(Процесс);
    B --> C{Решение};
    C -->|Да| D[Конец];
    C -->|Нет| E[Другой процесс];
    E --> B;
```

---

#### Диаграмма последовательности (Sequence Diagram)

**Шаблон:**
```mermaid
sequenceDiagram
    participant Алиса
    participant Боб
    Алиса->>Боб: Привет, Боб, как дела?
    Боб-->>Алиса: Отлично, Алиса! А у тебя?
    Алиса-xБоб: Я тоже в порядке!
```

**Результат:**
```mermaid
sequenceDiagram
    participant Алиса
    participant Боб
    Алиса->>Боб: Привет, Боб, как дела?
    Боб-->>Алиса: Отлично, Алиса! А у тебя?
    Алиса-xБоб: Я тоже в порядке!
```

---

#### Круговая диаграмма (Pie Chart)

**Шаблон:**
```mermaid
pie
    title Распределение задач
    "Документация" : 30
    "Разработка" : 50
    "Тестирование" : 20
```

**Результат:**
```mermaid
pie
    title Распределение задач
    "Документация" : 30
    "Разработка" : 50
    "Тестирование" : 20
```

У Mermaid есть еще множество других типов диаграмм. Подробнее можно почитать в [официальной документации](https://mermaid.js.org/syntax/classDiagram.html).
