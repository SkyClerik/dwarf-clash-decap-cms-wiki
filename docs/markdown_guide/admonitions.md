# Блоки предупреждений (Admonitions)

**Admonitions** — это расширение, которое позволяет создавать красивые, цветные блоки для выделения важной информации: заметок, предупреждений, советов и т.д.

---

### Как использовать

Вы начинаете блок с `!!!`, затем указываете тип блока (например, `note`, `warning`, `tip`) и, при желании, заголовок в кавычках. Весь текст внутри блока должен иметь отступ в четыре пробела.

### Примеры

#### Заметка (note)

**Шаблон:**
```markdown
!!! note "Это заголовок заметки"
    Здесь находится какой-то очень важный текст, который нужно знать читателю.
    Он будет оформлен в виде синего блока.
```

**Результат:**
!!! note "Это заголовок заметки"
    Здесь находится какой-то очень важный текст, который нужно знать читателю.
    Он будет оформлен в виде синего блока.

---

#### Предупреждение (warning)

**Шаблон:**
```markdown
!!! warning "Осторожно!"
    Это очень опасное действие, будьте внимательны!
    Этот блок будет желтого цвета.
```

**Результат:**
!!! warning "Осторожно!"
    Это очень опасное действие, будьте внимательны!
    Этот блок будет желтого цвета.

---

#### Совет (tip)

**Шаблон:**
```markdown
!!! tip
    Вот вам полезный совет по использованию нашего ПО.
```

**Результат:**
!!! tip
    Вот вам полезный совет по использованию нашего ПО.

---

### Все доступные типы

Вы можете использовать множество разных типов блоков:

**Шаблон:**
```markdown
!!! note "note"
    ...
!!! abstract "abstract"
    ...
!!! info "info"
    ...
!!! tip "tip"
    ...
!!! success "success"
    ...
!!! question "question"
    ...
!!! warning "warning"
    ...
!!! failure "failure"
    ...
!!! danger "danger"
    ...
!!! bug "bug"
    ...
!!! example "example"
    ...
!!! quote "quote"
    ...
```

**Результат:**

!!! note "note"
    ...
!!! abstract "abstract"
    ...
!!! info "info"
    ...
!!! tip "tip"
    ...
!!! success "success"
    ...
!!! question "question"
    ...
!!! warning "warning"
    ...
!!! failure "failure"
    ...
!!! danger "danger"
    ...
!!! bug "bug"
    ...
!!! example "example"
    ...
!!! quote "quote"
    ...
