# Вкладки (Tabs)

**Tabs** — это расширение, которое позволяет группировать контент по вкладкам. Это очень удобно, когда нужно показать схожую информацию для разных контекстов, например, примеры кода на разных языках программирования или инструкции для разных операционных систем.

---

### Как использовать

Вы начинаете блок с `===`, затем в кавычках пишете название вкладки. Весь контент, относящийся к этой вкладке, должен идти ниже с отступом в четыре пробела. Блоки разных вкладок должны идти друг за другом.

### Примеры

#### Простой пример

**Шаблон:**
```markdown
=== "Python"
    ```python
    print("Привет, мир!")
    ```

=== "JavaScript"
    ```javascript
    console.log("Привет, мир!");
    ```
```

**Результат:**
=== "Python"
    ```python
    print("Привет, мир!")
    ```

=== "JavaScript"
    ```javascript
    console.log("Привет, мир!");
    ```

---

#### Вложенные вкладки

Вы можете вкладывать вкладки друг в друга, создавая более сложную структуру.

**Шаблон:**
```markdown
=== "Инструкции по установке"

    === "Windows"
        Инструкция для Windows...

    === "macOS"
        Инструкция для macOS...

    === "Linux"
        Инструкция для Linux...

=== "Инструкции по использованию"
    Здесь идут инструкции по использованию...
```

**Результат:**
=== "Инструкции по установке"

    === "Windows"
        Инструкция для Windows...

    === "macOS"
        Инструкция для macOS...

    === "Linux"
        Инструкция для Linux...

=== "Инструкции по использованию"
    Здесь идут инструкции по использованию...
